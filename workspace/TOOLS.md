# Available Tools

OpenClaw provides these tools untuk workflow kamu:

## Development Tools

| Tool | Use Case |
|------|----------|
| `bash` | Execute commands, run scripts, development tasks |
| `read` | Read files, examine code |
| `write` | Create/write files |
| `edit` | Modify existing files |
| `glob` | Search files by pattern |
| `grep` | Search file contents |

## Research Tools

| Tool | Use Case |
|------|----------|
| `websearch` | Search web untuk information |
| `webfetch` | Fetch specific URLs |
| `codesearch` | Search code examples dari Exa |

## Browser Tools

| Tool | Use Case |
|------|----------|
| `browser` | Web automation, screenshots, form filling |

## Usage Notes

### Programming Tasks
- Use `bash` dengan `pty:true` untuk interactive CLIs
- Use `read` antes de modify files
- Verify changes avec `bash` command tests

### Marketing Tasks
- Use `websearch` untuk trend research
- Use `browser` untuk competitor analysis
- Use `webfetch` untuk grabbing content

## Best Practices

1. **Always verify** sebelum reporting completion
2. **Use version control** para projetos
3. **Document decisions** para future reference
4. **Test thoroughly** antes de deployment
5. **Measure results** para optimization
