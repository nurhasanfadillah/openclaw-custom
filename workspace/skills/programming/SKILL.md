# Programming

## Capabilities
- Development (full-stack, APIs, database)
- Debugging (error analysis, logs, tests)
- Code review (best practices, security)
- Refactoring (modernization, documentation)

## Tools
- `bash` - Execute commands
- `read/write/edit` - File operations
- `glob/grep` - Search code
- `websearch` - Research

## Workflow
1. Understand: requirements & context
2. Plan: approach & structure
3. Implement: clean, documented code
4. Test: verify functionality
5. Review: check quality

## Best Practices
- Read existing code before editing
- Use version control (git)
- Write tests alongside code
- Handle errors explicitly
