# OpenClaw Workspace

Custom workspace untuk **Digital Marketing & Pemrograman**.

## Quick Install

```bash
# Clone
git clone https://github.com/nurhasanfadillah/openclaw-custom.git

# Salin workspace
cp -r workspace/. ~/.openclaw/workspace
```

## Config

Edit `~/.openclaw/openclaw.json`:

```json
{
  "agents": {
    "defaults": {
      "workspace": "workspace",
      "model": "anthropic/sonnet-4-6"
    }
  }
}
```

## Tools

- `bash` - Execute commands
- `read/write/edit` - File ops
- `glob/grep` - Search
- `websearch/webfetch` - Research

## Skills

- `digital-marketing` - Strategy, content, analytics
- `programming` - Development, debugging, code review
